# 🧠 Step 1 — Get the transcript (huge time saver)

Open the video on YouTube
→ Click **“…” → Show transcript** → Copy it

Paste into ChatGPT (or any AI) and ask:

```text
List all track & field events mentioned with approximate timestamps (100m, 200m, long jump, javelin, etc.)
```

👉 This gives you a **first pass of events without watching everything**.

---

# 🧠 Step 2 — Identify athletics event blocks

In athletics videos, events follow patterns. Look for:

### 🏃 Track events

* “On your marks” → start
* Gunshot → race start
* Final sprint → finish

### 🏃‍♂️ Field events

* Long jump → run-up → takeoff → landing
* Javelin → run → throw release
* Shot put → rotation → release

👉 Each of these = **highlight candidate**

---

# ⚡ Step 3 — Quick video scan (don’t watch fully)

* Play at **2× speed**
* Jump every **2–3 minutes**
* Stop only when:

  * Crowd noise spikes
  * Athletes sprint or jump
  * Camera zooms / slow-motion replay

👉 Note timestamps only (10–15 total is enough)

---

# 🧠 Step 4 — Select BEST moments (very important)

Don’t include everything. Pick:

### 🔥 High-impact moments

* Race starts (explosive)
* Final 10–20 meters (finish)
* Big jump / throw
* Winner reaction / scoreboard

👉 Aim for **6–10 clips max**

---
